# Tests

This folder contains automated tests for `ServerMultiTool`.

## Projects
- `ServerMultiTool.NavigationTests`: unit tests for `NavigationService` and startup navigation behavior.
- `ServerMultiTool.UiAutomationTests`: smoke/UI automation style tests for shell navigation and startup tab selection.

## Run
```powershell
cd c:\Repositories\SaintZet\ServerMultiTool
dotnet test
```
